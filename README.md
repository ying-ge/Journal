# 期刊查询系统

基于自定义期刊认定目录的期刊评价查询工具，提供快速便捷的期刊水平查询服务。集成 JCR 影响因子（JIF）和 SCImago SJR 等国际权威期刊指标，帮助科研人员全面了解期刊信息。

## ✨ 核心功能

### 📊 期刊查询
- **自然科学类期刊查询**：涵盖 SCI 分区期刊（顶级期刊、Q1-Q3区等）
- **人文社科类期刊查询**：涵盖 SSCI 分区期刊（Q1-Q4区）
- **模糊搜索**：支持部分关键词快速匹配
- **分类浏览**：点击图例可查看该分类下的所有期刊

### 🔬 研究领域筛选 ⭐ NEW
- **10大研究领域**：医学、生物学、化学、物理、数学、地球科学、工程、农业、人文社科、艺术
- **智能识别**：根据期刊名称自动判断研究领域
- **快速筛选**：点击研究领域标签即可筛选相关期刊
- **图标显示**：每个领域配有专属图标，直观易识别

### 🏆 水平分类筛选 ⭐ NEW
- **9大水平分类**：
  - 🌟 国际顶级期刊（Nature、Cell、Science等）
  - 🥇 SCI Q1区A类、🥈 B类、🥉 C类、🏅 D类
  - 📊 SCI Q2区、📈 Q3区
  - 🎯 卓越行动计划
  - 🚀 高起点新刊
- **组合筛选**：支持与研究领域筛选叠加使用，取交集
- **精准定位**：快速找到特定水平的期刊
- **一键清除**：支持快速清除筛选条件

### 📖 OA期刊识别 ⭐ NEW
- **OA类型分类**：
  - 🟢 **完全OA**：所有文章开放获取，免费阅读
  - 🟡 **混合OA**：部分文章开放获取
  - ⚪ **传统期刊**：需要订阅才能阅读
- **智能判断**：基于DOAJ标准，识别准确率高
- **实时显示**：在JIF指标旁显示OA类型标识
- **帮助决策**：快速判断期刊是否开放获取

### 🔀 多选对比功能
- **勾选期刊**：在每个检索结果前勾选感兴趣的期刊
- **一键对比**：选中 2 个及以上期刊后，点击"📊 对比"按钮生成对比报告
- **清空选择**：点击"🗑️ 清空"按钮快速清除所有已选期刊
- **全面对比**：对比报告包含 17 个关键指标

### 🎯 权威指标集成

#### JCR 影响因子 (JIF) 指标
- **JIF (2024)**：2024年期刊影响因子
- **5-Year JIF**：五年影响因子
- **JIF Quartile**：JIF分区（Q1/Q2/Q3/Q4）
- **JIF Rank**：精确排名（如：2/135）
- **JCI (Journal Citation Indicator)**：标准化引用指标，可跨学科对比（世界平均=1.0）
- **无自引JIF**：去除自引用的影响因子，更真实反映期刊影响力
- **总引用次数**：期刊历史总被引次数
- **Publisher**：出版社信息
- **期刊缩写 & ISSN**：用于论文引用和文献管理

#### SCImago SJR 指标
- **SJR值**：SCImago 期刊排名指标
- **SJR Quartile**：SJR分区
- **Total Docs. (2024)**：2024年总文档数
- **APC (文章处理费)**：版面费信息，帮助投稿决策

## 🚀 使用方式

### 本地运行
由于浏览器安全限制，需要通过 HTTP 服务器运行：

```bash
# Python 3
python3 -m http.server 8000

# 或使用 Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器中访问：`http://localhost:8000`

### GitHub Pages 部署
项目已支持 GitHub Pages 部署，可直接在线访问使用。

### 使用研究领域筛选
1. 在搜索框下方找到"🔬 研究领域筛选"
2. 点击任一研究领域标签（如"🏥 医学与健康"）
3. 系统自动显示该领域的所有期刊
4. 点击"✕ 清除筛选"恢复显示所有期刊

### 使用水平分类筛选
1. 在搜索框下方找到"🏆 水平分类筛选"
2. 点击任一水平分类标签（如"🥇 SCI Q1区A类"）
3. 系统自动显示该分类的所有期刊
4. 可与研究领域筛选组合使用（如筛选"医学与健康"领域的"SCI Q1区A类"期刊）
5. 点击"✕ 清除筛选"恢复显示所有期刊

### 使用对比功能
1. 搜索期刊
2. 勾选感兴趣的期刊（至少2个）
3. 点击右下角"📊 对比"按钮
4. 查看生成的对比报告（包含研究领域、OA类型、水平分类等17项指标）
5. 点击"🗑️ 清空"可重新选择

## 📚 数据来源

- **期刊水平分类**：自定义期刊认定目录（自定义JCR分区与卓越行动计划评价体系）
- **JCR 指标数据**：2025年最新JCR完整版，网友分享（基于 Clarivate Analytics Journal Citation Reports）
- **SJR 指标数据**：SCImago Journal & Country Rank (基于 Scopus 数据)
- **OA期刊数据**：基于DOAJ (Directory of Open Access Journals) 标准的出版商和期刊识别
- **卓越行动计划期刊**：中国科技期刊卓越行动计划入选期刊目录

## 💡 指标说明

### OA期刊类型 ⭐
- **完全OA**：所有文章都开放获取，读者可免费阅读、下载、分享
- **混合OA**：部分文章开放获取（通常作者付费选择OA发表）
- **传统期刊**：需要订阅或付费才能阅读
- **识别方法**：基于DOAJ标准，通过出版商和期刊名智能判断

### JCI 指标
- 世界平均值为 1.0
- > 1.0 表示高于世界平均水平
- < 1.0 表示低于世界平均水平
- **唯一可跨学科比较的指标**

### APC (文章处理费)
- 显示期刊发表费用
- 帮助科研人员制定投稿预算
- 区分免费和收费期刊
