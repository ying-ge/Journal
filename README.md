# 期刊查询系统

基于单位内部期刊认定目录的期刊评价查询工具，提供快速便捷的期刊水平查询服务。集成 JCR 影响因子（JIF）和 SCImago SJR 等国际权威期刊指标，帮助科研人员全面了解期刊信息。

## ✨ 核心功能

### 📊 期刊查询
- **自然科学类期刊查询**：涵盖 SCI 分区期刊（顶级期刊、Q1-Q3区等）
- **人文社科类期刊查询**：涵盖 SSCI 分区期刊（Q1-Q4区）
- **模糊搜索**：支持部分关键词快速匹配
- **分类浏览**：点击图例可查看该分类下的所有期刊

### 🔀 多选对比功能 ⭐ NEW
- **勾选期刊**：在每个检索结果前勾选感兴趣的期刊
- **一键对比**：选中 2 个及以上期刊后，点击"📊 对比"按钮生成对比报告
- **清空选择**：点击"🗑️ 清空"按钮快速清除所有已选期刊
- **对比报告**：在弹窗中查看所有指标的横向对比表格

### 🎯 权威指标集成

#### JCR 影响因子 (JIF) 指标
- **JIF (2024)**：2024年期刊影响因子
- **5-Year JIF**：五年影响因子
- **JIF Quartile**：JIF分区（Q1/Q2/Q3/Q4）
- **JIF Rank**：精确排名（如：2/135）
- **JCI (Journal Citation Indicator)**：标准化引用指标，可跨学科对比（世界平均=1.0）
- **无自引JIF**：去除自引用的影响因子，更真实反映期刊影响力
- **总引用次数**：期刊历史总被引次数
- **Publisher**：出版社信息
- **期刊缩写 & ISSN**：用于论文引用和文献管理

#### SCImago SJR 指标
- **SJR值**：SCImago 期刊排名指标
- **SJR Quartile**：SJR分区
- **Total Docs. (2024)**：2024年总文档数
- **APC (文章处理费)**：版面费信息，帮助投稿决策

## 🚀 使用方式

### 本地运行
由于浏览器安全限制，需要通过 HTTP 服务器运行：

```bash
# Python 3
python3 -m http.server 8000

# 或使用 Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器中访问：`http://localhost:8000`

### GitHub Pages 部署
项目已支持 GitHub Pages 部署，可直接在线访问使用。

### 使用对比功能
1. 搜索期刊
2. 勾选感兴趣的期刊（至少2个）
3. 点击右下角"📊 对比"按钮
4. 查看生成的对比报告
5. 点击"🗑️ 清空"可重新选择

## 📚 数据来源

- **期刊水平分类**：单位内部期刊认定目录（中科院JCR分区与卓越行动计划评价体系）
- **JCR 指标数据**：2025年最新JCR完整版，网友分享（基于 Clarivate Analytics Journal Citation Reports）
- **SJR 指标数据**：SCImago Journal & Country Rank (基于 Scopus 数据)
- **卓越行动计划期刊**：中国科技期刊卓越行动计划入选期刊目录

## 💡 指标说明

### JCI 指标
- 世界平均值为 1.0
- > 1.0 表示高于世界平均水平
- < 1.0 表示低于世界平均水平
- **唯一可跨学科比较的指标**

### APC (文章处理费)
- 显示期刊发表费用
- 帮助科研人员制定投稿预算
- 区分免费和收费期刊
